# Business Performance Dashboard

## Project Overview

This project was developed to analyze pharmaceutical sales performance and provide business insights through an interactive dashboard.

The dashboard enables users to monitor key performance indicators (KPIs), evaluate sales trends, compare department performance, and identify high-performing products, hospitals, and sales representatives.

By automating data aggregation and visualization, the dashboard reduces manual reporting efforts and supports faster business decision-making.

---

## Key Features

### Executive KPI Dashboard
- Total Sales Performance
- Target Achievement Rate
- Sales Trend Monitoring
- Performance Summary Metrics

### Sales Performance Analysis
- Monthly Sales Trend Analysis
- Department Performance Comparison
- Product Performance Tracking
- Sales Representative Performance Analysis

### Interactive Dashboard
- Dynamic Filtering
- Drill-Down Analysis
- Interactive Charts and Tables
- Real-Time Data Exploration

### Reporting & Export
- Excel Export Functionality
- CSV Export Validation
- Automated Business Reporting

## Business Value

This dashboard helps management teams:
