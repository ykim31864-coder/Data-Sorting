# Pharmaceutical Sales Analytics Dashboard

## Project Overview

This project was developed to transform pharmaceutical sales data into a structured analytics platform using a complete end-to-end data pipeline.

Raw sales data was processed and transformed into a star schema data warehouse stored in SQLite. The structured data was then visualized through an interactive Dash dashboard to support business intelligence and performance analysis.

---

## Project Architecture

↓

↓

Star Schema Data Warehouse (SQLite)

↓

Interactive Dashboard (Dash + Plotly)
