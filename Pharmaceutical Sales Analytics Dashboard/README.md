# Pharmaceutical Sales Analytics Dashboard

## Project Overview

This project was developed to transform pharmaceutical sales data into a structured business intelligence platform using an end-to-end analytics pipeline.

Raw sales data was processed through an ETL workflow, transformed into a Star Schema data warehouse stored in SQLite, and visualized using an interactive Dash dashboard.

The dashboard provides sales performance monitoring, target achievement analysis, drill-down exploration, and detailed reporting across departments, teams, employees, and hospitals.

↓

Interactive Analytics Dashboard (Dash + Plotly)
