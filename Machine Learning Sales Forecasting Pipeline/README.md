# Machine Learning Sales Forecasting Pipeline

## Project Overview

This project was developed to forecast future pharmaceutical sales using machine learning techniques.

The pipeline automates data preprocessing, feature engineering, model training, evaluation, and forecasting to support business planning and sales strategy decisions.

By leveraging historical sales data, the model generates future sales predictions and provides performance metrics to evaluate forecasting accuracy.

---

## Key Features

### Sales Forecasting
- 2026 Sales Prediction
- Product-Level Forecasting
- Team-Level Forecasting
- Hospital-Level Forecasting

### Machine Learning Pipeline
- Data Preprocessing
- Feature Engineering
- Model Training
- Forecast Generation

### Forecast Validation
- Prediction Accuracy Evaluation
- Forecast Validation Dashboard
