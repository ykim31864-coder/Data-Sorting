# Machine Learning Sales Forecasting Pipeline

## Project Overview

This project was developed to forecast future pharmaceutical sales using machine learning techniques.

The pipeline automates data preprocessing, feature engineering, model training, evaluation, and forecasting to support business planning and sales strategy decisions.

By leveraging historical sales data, the model generates future sales predictions and provides performance metrics to evaluate forecasting accuracy.

---

## Key Features

### Data Processing
- Sales Data Cleaning
- Missing Value Handling
- Feature Engineering
- Time-Based Feature Creation

### Machine Learning Models
- Ridge Regression
- Random Forest Regression
- Model Comparison
- Automated Training Pipeline

### Forecasting
- Future Sales Prediction
- Trend Projection
- Monthly Sales Forecasting
- Forecast Visualization

### Model Evaluation
- Holdout Validation
- MAPE (Mean Absolute Percentage Error)
- Model Performance Comparison
- Prediction Accuracy Assessment
